🤖 Live model · G4 Predictor

Will this DNA fold into a G-quadruplex?

I trained and deployed this model myself. Type in a DNA string (A, C, G, T) and it returns a verdict with a probability score for whether the sequence forms a G-quadruplex, a four-stranded structure that matters for telomeres, gene regulation, and the kind of biology I study. It runs on Hugging Face and is embedded right here.
